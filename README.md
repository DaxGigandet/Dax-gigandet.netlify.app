# Dax Gigandet Official Website

This is the official website for artist Dax Gigandet. The site includes pages for biography, albums, merchandise, and contact information.

## Deployment

The site is designed for deployment on [Netlify](https://www.netlify.com/).

## Structure

- `index.html`: Home page with latest updates and welcome message
- `bio.html`: Artist biography
- `album.html`: Albums and discography
- `merch.html`: Merchandise store
- `contact.html`: Contact form and info
- `assets/css/styles.css`: Main stylesheet
- `assets/images/`: Image assets used across the site

## Contact Form

The contact form uses [Formspree](https://formspree.io/) for submissions. Replace the form action URL with your own Formspree endpoint.
